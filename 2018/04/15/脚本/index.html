脚本

发表于 2018-04-15

脚本就是给计算机照着做的

1、作业的两个脚本

a.demo.sh xxx 可在当前目录下生成目录 xxx，demo.sh yyy 可生成目录 yyy
b.如果要生成的目录已经存在，则直接退出
c.生成的目录结构如下：
├── css
│ └── style.css
├── index.html
└── js
└── main.js

![](/images/图片1523769454.jpg)
-d 表示存在的意思  $1 表示传入的第一个参数
![](/images/图片1523769523.jpg)

2、将 ~/local 添加到 PATH 里

a.cd ~/local; pwd 得到 local 的绝对路径
b.创建 ~/.bashrc：touch ~/.bashrc
c.编辑 ~/.bashrc：start ~/.bashrc，在最后一行添加 export PATH="local的绝对路径:$PATH”
d.source ~/.bashrc
之前你要运行 sh ~/local/demo.txt，现在你只需要运行 demo.txt 就行了（想想为什么，道理显而易见）

3、demo.txt 的后缀 .txt 很无聊，删掉它

mv ~/local/demo.txt ~/local/demo
现在你只要运行 demo 就能执行该脚本了
