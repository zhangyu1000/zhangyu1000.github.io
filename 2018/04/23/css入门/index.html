3、导航

a、去掉导航前面的点<ul style="list-style: none;">
去掉标签的默认样式<ul style="margin: 0;padding: 0;">

在css文件中
b、.topNavBar>nav>ul>li>a{}要想生效一定要和html标签的顺序严格对应

c、使导航栏横向
方法为给
html中<ul class="clearfix">,
css中

然后下面是clearfix的css
.clearfix::after{
content: “”;
display: block;
clear: both;
}
它的功能是防止出现bug

d、 .topNavBar>nav>ul>li>a{
改变a标签的字体大小 font-size: 12px;
字体加粗 font-weight: bold;
改变字体颜色 color: #；
去掉下划线 text-decoration: none；
}
给标签间距 .topNavBar>nav>ul>li{
margin-left: 17px;
margin-right: 17px;
}

e、 让标签点击出现边框.topNavBar>nav>ul>li>a:hover{
boder-bottem:1px solid red;
}
但这样写边框会让间距改变，则我们需要先给一个透明边框则不会：
.topNavBar>nav>ul>li>a{
border-bottem:1px solid transparent;
}
f、 为了让li包裹住a
.topNavBar>nav>ul>li>a{
display:block;
}

改变字体时需记得加上范围，例如
.topNavBar .logo .card{} 

body有固定的margin，所以需要给margin：0；消除掉，再自己手动控制添加padding
